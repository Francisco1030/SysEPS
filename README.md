#### Dependências
- NodeJs
- Express
- Angular CLI
- PG (PostgreSQL) ou Docker

------------

#### Comandos NodeJS
###### Baixar dependência
```shell
    npm install
```
###### Executar Aplicação
```shell
    npm dev || npm start
```

------------

#### Comandos Angular
###### Baixar dependência
```shell
    npm install
```
###### Executar Aplicação
```shell
    ng s
```

------------

#### Comandos Docker - DB
######   Verificar
```shell
sudo docker-compose config
```
######   Rodar
```shell
sudo docker-compose up -d
```
######   Parar
```shell
sudo docker-compose stop
```
